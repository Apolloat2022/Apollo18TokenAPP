# Apollo18 Token App

A modern cross-platform application for digital asset management.

## 🚀 Features

- Token reservation system with real-time tracking
- Multi-wallet integration  
- Cross-platform compatibility (iOS, Android, Web)
- Automated email notifications

## 🛠 Built With

- **React Native** - Cross-platform framework
- **Expo** - Development platform and tools
- **TypeScript** - Type safety
- **Modern web3 libraries** - Blockchain integration

## 📱 Platforms

- 📱 iOS
- 🤖 Android  
- 🌐 Web
